Lotería Mexicana

"LOTERIA ELECTRONICA. Teniendo un arreglo del 1 al 54 con los nombres de las cartas de lotería, escriba las funciones necesarias (pseudocódigo o utilice el lenguaje de su preferencia) para que imprima de manera aleatoria dichos nombres cada vez que se ejecute. Cada vez que se ejecuta, NO debe imprimir mas de 1 vez el nombre de una misma carta, es decir, cada que se ejecute la función deberán aparecer todas y cada una de las 54 cartas."
